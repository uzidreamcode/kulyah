Postion Static

postion static digunakan untuk mengatur element menjadi statis secara default. elemen akan mengikuti posisi normal secara default. elemen tidak dipengaruhi oleh properi seperti top,bottom, left dan right.

elemen ini diatur menggunakan posisi: static




Postion Relative

postion relative digunakan untuk mengatur element menjadi relative terhadap posisi normal. elemen akan mengikuti posisi normal secara default. elemen tidak dipengaruhi oleh properi seperti top,bottom, left dan right.

elemen ini diatur menggunakan posisi: relative




postion fixed

sebuah elemen dengan posisi fixed akan selalu berada pada posisi yang ditentukan. elemen dengan posisi fixed tidak akan bergerak saat pengguna melakukan scroll.

elemen ini diatur menggunakan posisi: fixed




















postion absolute

element html yang menggunakan posisi absolute akan diposisikan relative dengan elemen lain yang mendahuluinya yang terdekat. bukan relatif terhadap layar secara normal.

elemen ini diatur menggunakan posisi: relative
elemen ini diatur menggunakan posisi: absolute








posision sticky

cobalah untuk scroll untuk mengetahui posisi sticky berkerja.

catatan IE/Edge 15 versi yang lebih lama tidak mendukung sticky

elemen ini diatur menggunakan posisi: sticky

dalam contoh ini elemen sticky menempel pada posisi atas halaman (top:0), ketika mencapai posisi scrollnya
